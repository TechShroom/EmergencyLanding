EmergencyLanding

A game engine for Java games. Open source, because why not?

Built on OpenGL 3.2, uses modern OpenGL.

System Requirements:

  • Mac OSX:
    • OpenGL 3.2
    • OSX 10.7+ (as a result of requiring 3.2)
    • Any graphics card driver after 2009
  • Windows:
    • OpenGL 3.2
    • Any graphics card driver after 2009
  • Linux
    • OpenGL 3.2
    • Any graphics card driver after 2009
